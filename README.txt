# Sudoku Game 🎮

A web-based Sudoku game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Sudoku puzzle generator
- Difficulty levels
- Timer system
- Wrong move tracking
- Highlight selected numbers
- Interactive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
